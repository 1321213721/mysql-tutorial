# 安装/配置 MySQL 数据库

课程前半部分使用单节点的 MysQL ，所以在 Linux 上直接安装一个 MySQL 即可，后半段使用 docker 来安装 MySQL 集群

在笔者的另一个笔记中详细记录了 如何安装一个 MySQL8

# 关闭 SELinux

vi /etc/selinux/config
SELINUX=disabled
1
2

# 在线安装MySQL数据库

  • 替换 yum 源

    curl -o /etc/yum.repos.d/CentOS-Base.repo mirrors.163.com/.help/CentOS7-Base-163.repo
    1
    yum clean all 
yum makecache
    1
    2

  • 下载 rpm 文件

    yum localinstall https://repo.mysql.com//mysql80-community-release-el7-1.noarch.rpm
    1

  • 安装 MySQL 数据库

    yum install mysql-community-server -y
    1

# 本地安装 MySQL 数据库

  • 把本课程 git 工程里共享的 MySQL 本地安装文件上传到 Linux 主机的 /root/mysql 目录

  • 执行解压缩

    tar xvf mysql-8.0.11-1.el7.x86_64.rpm-bundle.tar
    1

  • 安装依赖的程序包

    yum install perl -y
yum install net-tools -y
    1
    2

  • 卸载 mariadb 程序包

    rpm -qa|grep mariadb
rpm -e mariadb-libs-5.5.60-1.el7_5.x86_64 --nodeps
    1
    2

  • 安装 MySQL 程序包

    rpm -ivh mysql-community-common-8.0.11-1.el7.x86_64.rpm 
rpm -ivh mysql-community-libs-8.0.11-1.el7.x86_64.rpm 
rpm -ivh mysql-community-client-8.0.11-1.el7.x86_64.rpm 
rpm -ivh mysql-community-server-8.0.11-1.el7.x86_64.rpm
    1
    2
    3
    4

  • 修改 MySQL 目录权限

    chmod -R 777 /var/lib/mysql/
    1

  • 初始化 MySQL

    mysqld --initialize
chmod -R 777 /var/lib/mysql/*
    1
    2

  • 启动 MySQL

    service mysql/mysqld start
    1

  • 查看初始密码

    grep 'temporary password' /var/log/mysqld.log
    1

  • 登陆数据库之后，修改默认密码

    mysql -u root -p
alter user user() identified by "abc123456";
    1
    2

  • 允许远程使用 root 帐户

    UPDATE user SET host = '%' WHERE user ='root';
FLUSH PRIVILEGES;
    1
    2

  • 允许远程访问 MySQL 数据库（/etc/my.cnf）

    character_set_server = utf8
bind-address = 0.0.0.0
    1
    2

  • 开启防火墙 3360 端口

    firewall-cmd --zone=public --add-port=3306/tcp --permanent
firewall-cmd --reload
    1
    2
